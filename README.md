# RESTful API Node Server Boilerplate

Install the dependencies:

```bash
yarn install
```

Set the environment variables:

```bash
cp .env.example .env

Running locally:

```bash
yarn dev
```

Running in production:

```bash
yarn start
```


Linting:

```bash
# run ESLint
yarn lint

# fix ESLint errors
yarn lint:fix

# run prettier
yarn prettier

# fix prettier errors
yarn prettier:fix
```
